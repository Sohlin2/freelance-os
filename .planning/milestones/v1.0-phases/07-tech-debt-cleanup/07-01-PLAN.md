**Fix .mcp.json (item 5):**
The current URL `https://mcp.freelanceos.dev/mcp` points to an undeployed server. Replace the hardcoded URL with an environment variable reference so the URL is configurable at install time. Change the `url` field from:
```
"url": "https://mcp.freelanceos.dev/mcp"
```
to:
```
"url": "${user_config.FREELANCEOS_SERVER_URL}"
```

Also add `FREELANCEOS_SERVER_URL` to the userConfig in plugin.json so the user is prompted for it at install time.

**Fix plugin.json skills field (items 6/11):**
The current `skills` field is a string `"./skills/"`. Per the Claude Code plugin spec, skills should be an array of glob patterns. Change from:
```
"skills": "./skills/"
```
to:
```
"skills": ["./skills/*/SKILL.md"]
```

**Add FREELANCEOS_SERVER_URL to plugin.json userConfig:**
Add a new entry to the `userConfig` object:
```json
"FREELANCEOS_SERVER_URL": {
  "description": "FreelanceOS MCP server URL (e.g., https://mcp.freelanceos.dev/mcp)",
  "sensitive": false
}
```
