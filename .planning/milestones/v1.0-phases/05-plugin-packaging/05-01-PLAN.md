**2. `.mcp.json`** at package root (per D-01, D-05):

```json
{
  "mcpServers": {
    "freelance-os": {
      "type": "http",
      "url": "https://mcp.freelanceos.dev/mcp",
      "headers": {
        "Authorization": "Bearer ${user_config.FREELANCEOS_API_KEY}"
      }
    }
  }
}
```

CRITICAL: Use `${user_config.FREELANCEOS_API_KEY}` NOT `${FREELANCEOS_API_KEY}`. The `user_config.` prefix reads from collected userConfig values. Without it, Claude Code reads from environment variables (empty).
Use `"type": "http"` for Streamable HTTP transport, NOT `"type": "sse"` (deprecated).

**1. Create `scripts/build-plugin.js`** (per D-08, D-09):

Write a Node.js ESM script (the package has `"type": "module"`) that does these steps in order:

a) Define constants:
   - `ROOT` = `resolve(import.meta.dirname, '..')`
   - `SKILLS_SRC` = `join(ROOT, '.claude', 'skills')`
   - `SKILLS_DEST` = `join(ROOT, 'skills')`
   - `PLUGIN_DIR` = `join(ROOT, '.claude-plugin')`
   - `PLUGIN_JSON` = `join(PLUGIN_DIR, 'plugin.json')`
   - `MCP_JSON` = `join(ROOT, '.mcp.json')`

b) Define secret scan patterns array:
   - `/service_role/i`
   - `/SUPABASE_SERVICE_ROLE/i`
   - `/eyJ[A-Za-z0-9+/=]{20,}/` (base64 JWT prefix used by Supabase keys)
   - `/fos_live_[0-9a-f-]{36}/` (live FreelanceOS API key values)
   - `/sb_[a-z]+_[A-Za-z0-9]{20,}/` (Supabase project reference keys)

c) Implement `scanForSecrets(dir)` async function:
   - Use `readdir(dir, { withFileTypes: true, recursive: true })`
   - For each file entry, read content with `readFile(full, 'utf8')`
   - Test each secret pattern against content
   - If match found: `console.error('SECRET DETECTED in ${full} — pattern: ${pattern}')` and `process.exit(1)`

d) Implement `build()` async function:
   1. Log "Building FreelanceOS plugin..."
   2. Remove `skills/` dir: `await rm(SKILLS_DEST, { recursive: true, force: true })`
   3. Copy skills: `await cp(SKILLS_SRC, SKILLS_DEST, { recursive: true })`
   4. Validate plugin.json exists and is valid JSON: `JSON.parse(await readFile(PLUGIN_JSON, 'utf8'))`
   5. Validate .mcp.json exists and is valid JSON: `JSON.parse(await readFile(MCP_JSON, 'utf8'))`
   6. Run secret scan on `skills/` directory
   7. Run secret scan on `.claude-plugin/` directory
   8. Read and scan `.mcp.json` content for secret patterns
   9. Count skill directories: `const skills = await readdir(SKILLS_DEST)` then filter for directories
   10. If skill count < 5: `console.error('Expected at least 5 skill dirs, found ${skills.length}')` and `process.exit(1)`
   11. Log "Build complete. ${skills.length} skills ready. No secrets detected."

e) Call `build().catch(e => { console.error(e); process.exit(1); })`

Import list: `cp, rm, mkdir, readFile, readdir` from `node:fs/promises`, `join, resolve` from `node:path`.
